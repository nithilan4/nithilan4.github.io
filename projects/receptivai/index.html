Project Overview

As a co-founder and lead developer of ReceptivAI, I developed a comprehensive customer service platform specifically designed for fitness studios. The platform leverages AI language models and integrates with various external services to provide a seamless experience for both studio owners and their customers.

Key Features

  • Fully responsive full-stack web application built with modern technologies
  • Integrated AI-powered response system for customer inquiries
  • Role-based authorization system for different user types
  • Custom email client with Google API integration
  • Stripe billing integration with subscription management

Technical Details

Front-end Development
Developed with Next.js 13 and ChakraUI to create a responsive and intuitive user interface
Authentication & Authorization
Implemented Next-Auth for secure user authentication and a role-based authorization system
Email Integration
Built a custom email client using Google Client APIs integrated with LLMs to generate responses and modify schedules
Payment Processing
Integrated with Stripe API for billing, handling delinquent users, and enforcing limits for resources

Development Highlights

One of the key technical challenges was implementing the NextAuth.js authentication system to work seamlessly with our role-based authorization requirements. I designed a custom middleware solution that validates user roles and subscription status before allowing access to protected resources.

The AI integration was built with a customizable system that allows studio owners to tailor the LLM responses to match their brand voice and specific business needs, without requiring technical knowledge.
